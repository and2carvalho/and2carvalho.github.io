Projects

Products and tools I've built.

SEIF — Cryptographic Provenance for AI Artifacts

Ecosystem

An open-source protocol that answers: who produced this AI artifact, under which policy, with what hash chain, and when? Uses Ed25519 signing, OpenTimestamps anchoring, and a classification gate (PUBLIC / INTERNAL / CONFIDENTIAL) built into the AI agent loop — not bolted on after. Designed to complement the Model Context Protocol (MCP): while MCP carries context between tools, SEIF attests to the provenance and integrity of artifacts created from that context.
